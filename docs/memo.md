  현재 상태

  - Memo 테이블 필드: id, content, created_at, updated_at, user_id
  - 현재 편집 가능: content만
  - UI: 심플한 에디터 + 저장/취소 버튼

  추가할 정보들

  EntityDetailDrawer를 참고하여 다음 정보들을 추가하겠습니다:

  1. 생성일자 (created_at) - 📝 편집 가능 (날짜/시간 선택기)
  2. 수정일자 (updated_at) - 👁️ 읽기 전용
  3. 메모 ID - 👁️ 읽기 전용 (디버깅/개발용)
  4. 연결된 엔티티 목록 - 👁️ 읽기 전용 (카드 형태)
